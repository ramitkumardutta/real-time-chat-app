# Real-Time Chat Application

## 🚀 Overview
A scalable real-time messaging platform enabling low-latency communication between multiple users using WebSockets (Socket.IO).

## ✨ Features
- Real-time messaging with Socket.IO
- JWT-based authentication
- Online/offline user presence
- Persistent chat history
- Secure user sessions

## 🛠 Tech Stack
- Node.js, Express.js
- MongoDB
- Socket.IO
- JWT Authentication

## ⚡ Key Highlights
- Designed for low-latency communication
- Handles concurrent user connections efficiently
- Optimized for real-time message delivery
